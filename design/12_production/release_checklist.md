# Public Release Checklist

## Identity and fan-work status

- [ ] Current official Touhou fan-work guidelines reviewed and date recorded.
- [ ] Store/title/credits clearly identify an unofficial fan work.
- [ ] No implication of endorsement or official status.
- [ ] Product title/logo visually distinct from official releases.
- [ ] Storefront/distributor/price/demo/crowdfunding plans checked separately.

## Assets and rights

- [ ] No extracted official sprites, music, SFX, dialogue, endings, fonts, code, or screenshots.
- [ ] No unlicensed fan art, arrangements, translations, fonts, or designs.
- [ ] Every shipped asset has creator, source, license/permission, and credit record.
- [ ] Guest contracts include game, promotion, soundtrack, and update rights as needed.
- [ ] Music cue status all `SHIP`, not `REFERENCE`.
- [ ] Placeholder scanner passes.
- [ ] Credits generated and manually reviewed EN/JA.

## Content

- [ ] Main story and route flags complete.
- [ ] Canon/Fanon/Original audit complete.
- [ ] All deep routes preserve autonomy and boundary tests.
- [ ] All-ages/PG-13 review complete.
- [ ] Comfort alternatives tested.
- [ ] No generated draft lacks human EN/JA/character review.
- [ ] Dream Theatre/AU content clearly labeled.

## Localization

- [ ] Zero missing keys.
- [ ] EN editorial sign-off.
- [ ] JA native editorial sign-off.
- [ ] Kinsoku and unsupported glyph checks pass.
- [ ] Screenshot matrix pass at supported UI scales.
- [ ] Locale switch and save/load pass.
- [ ] Store copy, EULA/notice, credits, support text translated as planned.

## Input and accessibility

- [ ] Keyboard complete.
- [ ] Controller complete.
- [ ] Remapping complete.
- [ ] One-handed presets pass.
- [ ] Story preset completes all mandatory content.
- [ ] Low Motion and no-flash pass.
- [ ] Bullet density/speed/contrast/auto-bomb pass.
- [ ] Fighter simple inputs/hold guard/auto-face pass.
- [ ] Audio mono and low dynamic-range pass.
- [ ] Content comfort toggles preserve plot.

## Saves and stability

- [ ] Atomic save and backup recovery pass.
- [ ] Every public save schema migration pass.
- [ ] Demo/release compatibility policy documented.
- [ ] Rollback does not overwrite newer saves.
- [ ] 20-run main smoke suite pass.
- [ ] Long soak shows no growing memory trend.
- [ ] Controller disconnect, suspend/resume, focus loss pass.
- [ ] Crash logs contain no sensitive data.

## Performance

- [ ] Target hardware measured, not assumed.
- [ ] Ordinary scenes meet frame target.
- [ ] Danmaku/fighter/festival/Journal stress fixtures pass or documented settings adjusted.
- [ ] Load-time targets measured.
- [ ] Memory targets measured.
- [ ] Debug resources stripped.

## Packaging

- [ ] Clean-machine install/run/uninstall Windows.
- [ ] Clean-machine Linux test.
- [ ] macOS signing/notarization if supported.
- [ ] Version, content revision, save schema correct.
- [ ] Release notes and known issues ready.
- [ ] Support/error-code documentation ready.
- [ ] SHA-256 manifest generated.
- [ ] Previous build retained for rollback.
- [ ] Final package archived with test and rights reports.
