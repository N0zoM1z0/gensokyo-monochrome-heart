# Review Checklists

## Narrative event

- [ ] Clear contradiction/problem.
- [ ] Every lead has an immediate non-romantic motive.
- [ ] Comedy escalates through character/location logic.
- [ ] Mechanical mode expresses the conflict.
- [ ] Quiet afterbeat changes an object, habit, or boundary.
- [ ] Failure and assist branches are authored.
- [ ] Journal records observation rather than omniscience.
- [ ] Event length fits class budget.

## Character / canon

- [ ] `skills.md` read.
- [ ] Relationship graph checked.
- [ ] Official anchor not contradicted.
- [ ] Sparse canon not filled by unlabeled fanon.
- [ ] Fanon dial within ceiling.
- [ ] Character can refuse and leave.
- [ ] Character succeeds through own competence.
- [ ] Protagonist is not the only motive.
- [ ] Original dialogue not presented as canon.

## Romance / boundaries

- [ ] Intimacy matches Trust/Ease/Spark bands.
- [ ] Authority, medical care, intoxication, injury, fear, or compromised autonomy are not treated as consent.
- [ ] Boundary test has consequences.
- [ ] Relationship does not erase duties/friends/faction identity.
- [ ] Content remains PG-13.
- [ ] Comfort toggle alternative preserves information.

## Gameplay

- [ ] Controls introduced and remappable.
- [ ] Goal/failure clear without color.
- [ ] Story assists supported.
- [ ] No rapid-tap requirement unless optional and replaceable.
- [ ] Clear/excellent/loss/assist results typed.
- [ ] Restart resets all state.
- [ ] Performance fixture exists.
- [ ] Mechanic does not solve social choice automatically.

## UI / localization

- [ ] No hard-coded display text.
- [ ] EN/JA complete.
- [ ] Width/line-count pass.
- [ ] Japanese kinsoku and register review.
- [ ] Focus/cancel/controller pass.
- [ ] Nonnumeric relationship feedback.
- [ ] Low Motion/no-flash pass.

## Art

- [ ] Original design, no tracing/copying.
- [ ] Recognizable silhouette.
- [ ] Both polarities readable.
- [ ] Anchors/collision metadata correct.
- [ ] Dither does not flicker.
- [ ] Accessibility outline pass.
- [ ] Source/export/provenance recorded.

## Audio

- [ ] Cue purpose and transition states implemented.
- [ ] Rights status complete.
- [ ] Loop clean.
- [ ] Dialogue/warning hierarchy pass.
- [ ] Mono and low dynamic-range pass.
- [ ] SFX voice limits pass.
- [ ] No copyrighted meme/placeholder remains.

## Engineering

- [ ] Layer/dependency direction respected.
- [ ] Typed boundary.
- [ ] Tests positive/negative.
- [ ] Save/replay compatibility considered.
- [ ] No unapproved dependency.
- [ ] No per-frame avoidable allocations in hot path.
- [ ] Error path safe and diagnostic.
- [ ] Exact verification reported.
