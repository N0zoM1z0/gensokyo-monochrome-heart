# Production Package

This folder converts the design into a realistic staged production plan.

- `milestones.md` — sequence, gates, and indicative duration
- `content_budget.md` — asset/event counts and scope tiers
- `staffing_and_pipeline.md` — roles and handoffs
- `risk_register.md` — probability/impact/mitigation/trigger
- `release_checklist.md` — public-build gate
- `review_checklists.md` — narrative, character, art, audio, technical reviews
- `decision_record_template.md` — design change control
