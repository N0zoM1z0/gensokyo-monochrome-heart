# Route Structure

## Deep route template

Each deep route has seven nodes.

1. **Public mask** — encounter the character in their recognized role.
2. **Competence** — help with real work.
3. **Private habit** — discover what happens when no audience is present.
4. **Friction** — the player's usual tone stops working.
5. **Reciprocity** — the character helps the protagonist without performance.
6. **Crisis** — the Monochrome Incident offers a tempting permanent solution.
7. **Promise** — friendship or romance commitment shaped by the character's values.

## Route gates

A node checks facets rather than total points.

Example:
- Reimu node 4 requires Trust 2 and either Respect 2 or Ease 3.
- Sakuya node 5 requires the player to have allowed a task to remain imperfect.
- Aya node 6 requires at least one private fact withheld from publication.
- Kaguya node 6 changes if the player repeatedly accepted infinite retries.

## Route endings

Each route has:
- friendship ending;
- romance ending;
- delayed ending if the player postpones;
- strained ending if the crisis is resolved without repairing route conflict.

No route has a “bad ending” based on refusing romance.

## Cross-route scenes

Deep routes create pairs:
- Reimu / Marisa: effortless talent and deliberate effort
- Sakuya / Remilia: service, loyalty, and self-authorship
- Patchouli / Marisa: knowledge, borrowing, and intellectual respect
- Youmu / Yuyuko: duty and trust
- Aya / Hatate: public record and mediated observation
- Eirin / Kaguya: care, exile, and eternity
- Sanae / Reimu: faith and modern framing
- Tenshi / Shion: boredom, attention, and consequence

## Route concurrency

The player may advance all routes. The game does not pretend parallel romance is invisible:
- characters notice scheduling;
- the player must answer honestly;
- romance flags may coexist only after boundary conversations;
- the Ensemble Accord is not automatic.
