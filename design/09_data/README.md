# Machine-Readable Starter Data

This folder is a reviewed starting point for Codex and the content toolchain. It is deliberately smaller than the full prose design package.

- `characters.json` — runtime-facing metadata for 71 character profiles
- `locations.json` — 19 region records and map positions
- `events.json` — launch planning event index
- `sample_event_empty_cushion.json` — complete example event graph
- `dialogue_samples.json` — localized dialogue beat examples
- `localization/strings.csv` — bilingual string starter
- `schemas/` — JSON Schemas
- `music_cues.csv` — copy of the 89-row cue sheet
- `content_index.json` — versions and file inventory

All IDs are provisional until the repository is initialized. Once a public build ships, IDs must never be silently reused.
