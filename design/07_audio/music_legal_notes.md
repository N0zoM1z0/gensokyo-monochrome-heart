# Music Rights and Credits Workflow

This is a conservative production checklist, not legal advice.

## 1. What the cue sheet means

A Touhou theme listed in the cue sheet is a **creative reference and search aid**. It is not an included asset, a license, or permission to use someone else's recording.

## 2. Safe production options

1. Commission an original arrangement and recording under a written agreement.
2. Create the arrangement in-house from a clean-room transcription and original performance/production.
3. License an existing fan arrangement directly from its rights holder, while confirming that both the arrangement license and Touhou fan-work rules permit the intended game/platform use.
4. Replace a reference cue with a wholly original composition using only the project's regional motifs.

## 3. Never ship

- ripped game audio;
- music-room files extracted from official games;
- a fan arrangement downloaded from a video or album without permission;
- stems generated from someone else's master recording;
- unlicensed karaoke/MIDI files;
- a cover whose distribution terms do not include games;
- placeholder commercial tracks accidentally left in a build.

## 4. Composer brief packet

Each commission receives:
- cue ID and gameplay video;
- reference theme title and source work;
- permitted quotation/arrangement scope confirmed by producer;
- target length, loop points, stems, BPM, meter;
- instrument and chip constraints;
- accessibility/density requirements;
- delivery formats;
- credit wording;
- fee, revision count, ownership/license, soundtrack rights.

## 5. Credits database

Track per cue:
- original Touhou composition credit;
- source game/work;
- arranger;
- performer;
- recording/mix/master;
- license/permission document path;
- in-game title;
- soundtrack title;
- storefront attribution;
- content ID / automated claim notes.

## 6. Release gate

No cue can move from `REFERENCE` to `SHIP` until its rights record is complete. Run a final build scan for temporary WAV/OGG filenames and compare against the approved cue database. Recheck the current official Touhou fan-work guidelines before release and storefront submission.
