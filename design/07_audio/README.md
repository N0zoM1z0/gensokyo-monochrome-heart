# Audio Package

No audio files are included. This folder tells a composer or arranger what each part of the game needs and lists useful Touhou theme references for later sourcing or original arrangement work.

- `audio_direction.md` — sound identity and adaptive music system
- `music_cue_sheet.md` / `.csv` — scene-by-scene reference and composition brief
- `sfx_catalog.md` — gameplay, UI, environment, and character sound vocabulary
- `music_legal_notes.md` — conservative rights and credits workflow
