# Support Packet — Release Candidate

No public support channel is assigned for `0.1.0-rc.1`; this candidate is not
approved for public distribution. Before publishing, replace this statement
with an owned support address and response policy.

When a support channel is assigned, collect only:

- game version, content revision, save schema, OS, and renderer category;
- a stable error code and stack trace where the player explicitly chooses to
  send it; and
- accessibility configuration only when relevant to the problem.

Do not request or transmit a full save, protagonist name, personal filesystem
path, or free-form dialogue by default. Ask for a redacted reproduction path
first. See `design/08_technical/build_release.md` for the logging boundary.
