# Compact Fighter Runtime Sheets

These four runtime PNGs are byte-identical copies of the reviewed Model M
Reimu and Marisa sheets in
`design/06_art/visual_system_v2/assets/sprites/`. The `design/` tree is excluded
from Godot import by design, so the runtime copies live here. They remain native
24×32 key-pose placeholders; combat hurtboxes and hitboxes are independently
authored in `content/fighter/reimu_marisa_duel.json`.
